README speaktodo


speakTODO™ App - minimalist TODO AI-manager powered by voice

ОПИСАНИЕ Приложения

TodoList состоит из кусочков информации в виде слов. Это дела, которые нужно сделать. И они хранятся на сервере, например, в Postgre, базе данных или MySQL. И агент на этом сайте помогает в следующем. Например, я могу сказать, выведи мне список дел, которые запланированы, которые нужно сделать в марте 2025 года. Начни с самых важных, и в конце списки наиболее или менее важные. ИИ агент зайдет, будучи на сайте, он связывается с базой данных, найдет эти дела и выведет их на экран. Он способен связаться с базой данных, обработать запросы, и отобразить результаты в нужном порядке, используя алгоритмы для сортировки задач по приоритету.

ИИ-помощник может реализовать все перечисленные функции: добавление новых задач вручную, категоризацию, отображение или скрытие выполненных задач, удаление, а также фильтрацию по срочности.

DEVELOPMENT STEPS

Определиться с базой данных (например, PostgreSQL).
Выбрать язык программирования (например, JavaScript, Python).
Создать пользовательский интерфейс.
Интегрировать ИИ (например, с помощью API).
Разработать голосовое управление.
Протестировать и оптимизировать приложение.


Использовать платный аккаунт ChatGPT для создания API нельзя, поскольку API предоставляется отдельно. 
Подписка на ChatGPT не включает API. Для API потребуется отдельная подписка.
Для таких целей есть отдельные API-подписки. Что касается GPT-3 от OpenAI, бесплатный доступ не предоставляет API. Для интеграции ИИ в приложение нужно использовать платную подписку API.

Стоимость API от OpenAI в 2025 году зависит от модели: 
GPT-4o стоит примерно $2,50 за миллион входных токенов и $10 за миллион выходных. 
GPT-4o mini стоит $0,15 за миллион входных и $0,60 за миллион выходных токенов.​
GPT-3.5 turbo стоит $0,50 за миллион входных и $1,50 за миллион выходных токенов.​

Gemini 2.0 Flash стоит $0,10 за миллион входных и $0,40 за миллион выходных токенов.​
Модель Gemini 2.0 Flash-Lite стоит $0,075 за миллион входных и $0,30 за миллион выходных токенов.​

Исходя из того, что один человек создает в среднем 50-100 задач в месяц:
Для Gemini 2.0 Flash, стоимость будет примерно €0,30 за миллион выходных токенов. 
Если каждый создает 100 задач, и на задачу 10 токенов, то будет €1-3 в месяц.
Для GPT-3.5 Turbo примерно €1,50 за миллион выходных токенов. 
Это может быть €2-6 в месяц.
